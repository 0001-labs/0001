Settlementet

Draft for approval

Overview

Settlementet needed a practical migration path for a large Danish nonprofit website while preserving donation, newsletter, and support flows that already depended on external services.

Work

Mapped the current WordPress and Elementor setup, documented the page structure, reviewed the OnlineFundraising.dk iframe payment flow, and planned how the donation experience could move into Squarespace without rebuilding the payment system.

Outcome

The migration plan separated content work from payment infrastructure, reducing delivery risk and making it clear which parts could move directly and which needed careful testing.
