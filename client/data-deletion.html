Data deletion

Effective date: April 25, 2026

You can request deletion of personal data associated with 0001.dev at any time.

How to request deletion

  • Email joachim@0001.dev with the subject "Data deletion request" and include the account email or Meta app-scoped user ID, if available.
  • If you remove the app from Meta, Meta may send an automated callback to /api/meta/data-deletion.
  • If you received a confirmation code, include it in your email so we can check the queued request.

What we delete

We delete or anonymize personal account data, Meta-derived identifiers, access tokens, connection records, and feature data associated with the verified request, unless retention is required for law, security, abuse prevention, or accounting.

Timeline

Valid deletion requests are processed within 24 hours. Some operational logs or legally required records may remain for a limited period and are isolated from active product use.

For Meta

The Meta Data Deletion Request Callback URL is https://0001.dev/api/meta/data-deletion. The endpoint accepts POST requests with signed_request and returns a status URL plus confirmation_code.

Contact

For deletion requests or status checks, contact joachim@0001.dev.
